Changelog

Release history and notable updates
11 February 2026
Open-source release
First public release of WatchTurm Control Room. GitHub, TeamCity, and Jira integrations. Snapshot generator, Release History, Ticket Tracker, Statistics, Runbooks, Environment details with compare. Setup guide for DevOps.
2 February 2026
Runbooks: Scope & drift checkers
Scope Checker and Drift Checker runbooks. Compare baseline vs default branch, detect changes not yet merged. Ticket prefix filtering, clickable commit URLs.
22 January 2026
Statistics dashboard
Deployment frequency, lead time, top deployers per project and environment. Time window selector (7d / 30d). Deployment-focused analytics from release history.
15 January 2026
Release History append-only
Enterprise storage format: index.json + events.jsonl. 60-day backfill from snapshot. Calendar view and advanced filters.
8 January 2026
Ticket Tracker enhancements
Multi-PR ticket support, environment presence from deployments. Per-PR timelines for diverged rollouts. Jira summary and status integration.
28 December 2025
Environment compare mode
Select up to 2 environments, see component version mismatches. Visual diff for tags, branches, builds. Inline quick links (CMS, frontends) per environment.
18 December 2025
Snapshot engine core
GitHub + TeamCity + Jira integration. Kustomization discovery, component metadata, deployer info. Automated snapshot with scheduler.
2 December 2025
Overview & pipeline radar
Project slices (pizza wheel) with environment status. Click to expand details. Glass UI, dark & light theme. JSON-driven data model.
21 November 2025
Initial SPA structure
Lightweight single-page app. Left navigation: Overview, Platforms, Tools. Basic layout and routing. Exo 2 typography, responsive design.
14 November 2025
Project kickoff
WatchTurm Control Room initiative started. Release management dashboard for multi-platform environments.
Developed by Mateusz Zadrożny within the WatchTurm initiative and WatchTurm Control Room project.
